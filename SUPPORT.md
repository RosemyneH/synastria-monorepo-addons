# Support

- **Wrong or missing add-on link in the catalog:** open an [Issue](https://github.com/RosemyneH/synastria-monorepo-addons/issues) in this repository (or use the issue template).
- **Attune Helper Companion** (tray, install failures, game path, UI): use [AttuneHelperCompanion Issues](https://github.com/RosemyneH/AttuneHelperCompanion/issues).

The JSON list here is maintained for Synastria-style 3.3.5 add-ons; in-game or server issues should go to your server’s support channels.
