# Security policy

## Supported versions

Security fixes are applied to the default branch (`main`) of this repository and flow into the Attune Helper Companion release process as the catalog is updated.

## Reporting a vulnerability

**Please do not** file public issues for undisclosed security problems in the catalog or tooling.

- Prefer **GitHub** → this repository → **Security** → **Report a vulnerability** (if enabled for the repo).
- Alternatively, contact the maintainers through a private channel listed on the [AttuneHelperCompanion](https://github.com/RosemyneH/AttuneHelperCompanion) repository if this repository does not have private reporting enabled.

Include: affected files or URLs, steps to reproduce, and impact. We aim to acknowledge within a few business days.

This repository holds JSON and documentation; many issues are best reported on the companion app repository if they concern the desktop or mobile client.
